Sobre o Projeto

Este projeto foi desenvolvido como parte da disciplina de Engenharia de Software do curso de Engenharia da Computação do CEFET-MG, Campus Divinópolis.

O objetivo foi aplicar os conceitos aprendidos em sala de aula para desenvolver um sistema completo, desde a concepção até a implementação, utilizando metodologias ágeis e boas práticas de desenvolvimento de software.

A plataforma CineTickets foi criada para facilitar a reserva de ingressos de cinema, oferecendo uma experiência intuitiva e agradável aos usuários, enquanto fornece ferramentas poderosas de gerenciamento para funcionários.
